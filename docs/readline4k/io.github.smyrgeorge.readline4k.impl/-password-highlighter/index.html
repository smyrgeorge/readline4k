PasswordHighlighter

class PasswordHighlighter : Highlighter(source)

A simple Highlighter that masks all input characters as asterisks.

This highlighter is intended for password-like input. It replaces the visual representation of the current line with a sequence of '*' of the same length, keeping the actual input intact in the editor's buffer.

Behavior notes:

  • The highlight method always returns a masked string regardless of the cursor position.

  • The highlightChar method returns false for Highlighter.CmdKind.MOVE_CURSOR so that pure cursor movement does not trigger unnecessary re-rendering of characters; other command kinds return true to ensure the masked view stays up to date.
